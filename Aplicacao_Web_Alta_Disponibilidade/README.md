Neste laboratório, implantei a aplicação web TravelAgency em várias zonas de disponibilidade com um Application Load Balancer e health checks para fornecer alta disponibilidade. 
- Configurei um grupo de Auto Scaling para usar um Application Load Balancer.
- Configurei o health checks do load balancer para o Grupo Auto Scaling.
- Adicionei uma segunda zona de disponibilidade ao grupo de Auto Scaling.
- Configurei o grupo existente do Auto Scaling para incluir uma nova instãncia do EC2 em uma terceira zona de disponibilidade.
