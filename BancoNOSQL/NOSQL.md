

Nesse laboratório, implementei o Amazon DynamoDB para armazenar e processar dados de comportamento do espectador, incluindo consumo de conteúdo e análise de dispositivo.

- Criei um banco de dados NoSQL como uma tabela do Amazon DynamoDB.
- Adicionei registros, com esquema dinâmico, à tabela do DynamoDB.
- Consultei a tabela do DynamoDB.
- Criei um novo item na tabela UserVideoHistory com um valor de User ID exclusivo.
- Criei um atributo Number chamado "rating" no registro.
